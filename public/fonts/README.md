# Pretendard Variable 폰트

이 디렉토리에 `PretendardVariable.woff2` 파일을 넣어주세요.

## 다운로드

1. https://github.com/orioncactus/pretendard/releases 에서 최신 릴리즈 다운로드
2. `Pretendard-1.x.x.zip` 또는 `PretendardVariable.zip` 압축 해제
3. `PretendardVariable.woff2` (dynamic subset 권장) 파일을 이 디렉토리에 복사

```
public/fonts/PretendardVariable.woff2  ← 여기에 위치해야 함
```

> `.gitignore`에 `*.woff2`가 등록되어 있으므로 git에는 추적되지 않습니다.
> Vercel 배포 시 빌드 전에 폰트 파일을 포함시키거나, 별도 CDN에서 로드하세요.
